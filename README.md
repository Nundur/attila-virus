# ATTILA VIRUS

A self replicating mini virus built by me for testing, and learning how these type of viruses work (worms)

<hr/>


Features:
- Self-replicates and executes itself
- Encrypts files using XOR encryption
- Infects other .exe files with its own code and executes them
- Infects other file types with its own code and executes them
